# Personal Dotfiles

This repository contains my personal configuration files for syncing settings across my systems.

## Apps & Tools Configured

### Window Manager & Desktop

- **Hyprland** - Wayland compositor and window manager
- **Waybar** - Status bar for Wayland
- **Rofi** - Application launcher
- **Wlogout** - Logout menu with custom assets
- **Dunst** - Notification daemon

### Terminal & Shell

- **Kitty** - Terminal emulator
- **Fish** - Shell with Catppuccin themes
- **Bash** - Fallback shell configuration

### Theming

- **GTK 3/4** - GTK application theming
- **Qt5ct/Qt6ct** - Qt5/Qt6 configuration tools
- **Kvantum** - Qt theme engine (Catppuccin Mocha Rosewater)

### Other Tools

- **Fastfetch** - System information tool
- **Nano** - Text editor
- **VSCode** - Code editor

## Theme

Most configurations use the **Catppuccin** color scheme (mostly Mocha).
